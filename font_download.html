1. DilleniaUPC (ภาษาไทยปกติ)

สำหรับพิมพ์เนื้อหาภาษาไทย ออกข้อสอบวิชา แปลไทยเป็นมคธ

ดูกรภิกษุทั้งหลาย ความรักก็ดี ความชังก็ดี ย่อมเกิดจากของที่รัก...

2. DilleniaUPC_4Balee (ภาษาบาลี)

สำหรับพิมพ์เนื้อหาภาษาบาลี ออกข้อสอบวิชา แปลมคธเป็นไทย และ สัมพันธ์ไทย

ฟอนต์นี้มีการปรับแต่งพิเศษ: ตัดเชิง ญ และ ฐ ออก เพื่อความถูกต้องตามหลักบาลี

ธมฺมปทฏฺฐกถาย ปญฺจโม ภาโค (ตติโย วคฺโค)...

ดาวน์โหลดฟอนต์ทั้งหมด (Pack)

รวมฟอนต์ที่จำเป็นทั้งหมดในไฟล์เดียว (DilleniaUPC, DilleniaUPC_4Balee, Buddhadham, Buddhawajana, Burapa)

ดาวน์โหลด fonts.zip
วิธีการติดตั้งฟอนต์
1

ดาวน์โหลดไฟล์

กดปุ่ม ดาวน์โหลด fonts.zip ด้านบน และบันทึกไฟล์ลงในเครื่อง

2

แตกไฟล์ (Extract)

คลิกขวาที่ไฟล์ fonts.zip แล้วเลือก Extract All... (หรือ "แยกไฟล์ทั้งหมด")

3

ติดตั้ง

เข้าไปในโฟลเดอร์ที่แตกไฟล์ออกมา เลือกไฟล์ฟอนต์ทั้งหมด (Ctrl+A) คลิกขวา แล้วเลือก Install (หรือ "ติดตั้ง")

* หากมีหน้าต่างแจ้งเตือนว่ามีฟอนต์อยู่แล้ว ให้เลือก Replace เพื่ออัปเดตเป็นเวอร์ชันล่าสุด

1

ดาวน์โหลดไฟล์

กดปุ่ม ดาวน์โหลด fonts.zip ด้านบน ไฟล์จะไปอยู่ที่โฟลเดอร์ Downloads

2

เปิดไฟล์

ดับเบิ้ลคลิกที่ไฟล์ fonts.zip ระบบจะแตกไฟล์ออกมาเป็นโฟลเดอร์

3

ติดตั้งลง Font Book

เปิดโฟลเดอร์ฟอนต์ เลือกไฟล์ฟอนต์ทั้งหมด แล้วดับเบิ้ลคลิก โปรแกรม Font Book จะเปิดขึ้นมา ให้กดปุ่ม Install Font (ติดตั้งฟอนต์)
