ทิวาทิคณิกธาตุ (หมวด ทิว ธาตุ)

หมวดธาตุมี ทิวุ ธาตุเป็นต้น มี ย เป็นวิกรณปัจจัยในกัตตุวาจก เมื่อลงแล้วมีอำนาจดังนี้

1. หลักเกณฑ์ทั่วไป

ลง ย ปัจจัยหลังธาตุที่มีสระเดียวให้คง ย ไว้
  • ขี + ย + ติ = ขียติ
  • ฆา + ย + ติ = ฆายติ
  • วา + ย + ติ = วายติ
  • ตา + ย + ติ = ตายติ
  • ฌา + ย + ติ = ฌายติ
  • ยา + ย + ติ = ยายติ

2. กฎเฉพาะ ชน ธาตุ

เมื่อลง ย ปัจจัยท้าย ชน ธาตุ เอาพยัญชนะที่สุดธาตุ เป็น อา เช่น ชายติ

3. การเปลี่ยนแปลง ย ปัจจัย (เทฺวภาว)

เมื่อลง ย ปัจจัยแล้ว ย จะเปลี่ยนรูปตามพยัญชนะที่อยู่หน้า สำหรับธาตุที่ลงท้ายด้วยพยัญชนะ ต้องทำ เทฺวภาว (คือซ้อนอักษร) โดยทำนองเดียวกับในกัมมวาจกและภาววาจกด้วย (แต่ไม่ต้อง ลง อิ อาคม)

3.1 แปลง ย กับที่สุดธาตุ ก, ฏ, ป วรรค

แปลง ย ปัจจัย กับที่สุดธาตุซึ่งเป็นอักษร ใน กวรรค, ฏวรรค และ ปวรรค เป็นอักษรเหมือนกันกับพยัญชนะที่สุดธาตุของวรรคทั้ง 3 ดังกล่าวนั้น และเอาพยัญชนะอัพภาสซึ่งเป็นพยัญชนะที่ 2 และ 4 เป็นพยัญชนะที่ 1 และที่ 3 ของวรรคนั้นๆ

วรรค การเปลี่ยนแปลง ตัวอย่าง
วรรค ก ก+ย = กฺก
ข+ย = กฺข
ค+ย = คฺค
ฆ+ย = คฺฆ		 สก+ย+ติ = สกฺกติ
วรรค ฏ ฏ+ย = ฏฺฏ
ฐ+ย = ฏฺฐ
ฑ+ย = ฑฺฑ
ฒ+ย = ฑฺฒ
ณ+ย = ญฺญ		 ลุฏ+ย+ติ = โลฏฺฏติ
วรรค ป ป+ย = ปฺป
ผ+ย = ปฺผ
พ+ย = พฺพ
ภ+ย = พฺภ
ม+ย = มฺม		 ตป+ย+ติ = ตปฺปติ
ลุภ+ย+ติ = ลุพฺภติ
สมุ+ย+ติ = สมฺมติ

3.2 แปลง ย กับที่สุดธาตุ จ, ต วรรค

แปลง ย ปัจจัย กับที่สุดธาตุซึ่งเป็นอักษรใน จวรรค, ตวรรค เป็นอักษรเหมือนกันกับพยัญชนะที่สุดธาตุของวรรค จ และเอาพยัญชนะอัพภาสซึ่งเป็นพยัญชนะที่ 2 และ ที่ 4 เป็นพยัญชนะที่ 1 และที่ 3

วรรค การเปลี่ยนแปลง ตัวอย่าง
วรรค จ จ+ย = จฺจ
ฉ+ย = จฺฉ
ช+ย = ชฺช
ฌ+ย = ชฺฌ
ญ+ย = ญฺญ		 ปจ+ย+ติ = ปจฺจติ
ลุช+ย+ติ = ลุชฺชติ
วรรค ต ต+ย = จฺจ
ถ+ย = จฺฉ
ท+ย = ชฺช
ธ+ย = ชฺฌ
น+ย = ญฺญ		 นต+ย+ติ = นจฺจติ
ปท+ย+ติ = ปชฺชติ
พุธ+ย+ติ = พุชฺฌติ
หน+ย+ติ = หญฺญติ

3.3 ซ้อนพยัญชนะเศษวรรค (ร, ล, ว, ส, ห)

  • ร+ย = ยฺย (ปูรี+ย+ติ = ปุยฺยติ)
  • ล+ย = ลฺล (ปิล+ย+ติ = ปิลฺลติ)
  • ว+ย = พฺพ (ทิวุ+ย+ติ = ทิพฺพติ)
  • ส+ย = สฺส (มุส+ย+ติ = มุสฺสติ)
  • ห+ย = ยฺห (นห+ย+ติ = นยฺหติ)
นอกจากนี้แล้วเฉพาะ ทิวุ ธาตุ ยังทำตัวรูปสำเร็จด้วยวิกรณปัจจัยในหมวด จุราทิ ได้ด้วย
