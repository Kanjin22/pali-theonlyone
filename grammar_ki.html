ปิดหน้านี้

กิยาทิคณิกธาตุ (หมวด กี ธาตุ)

หมวดธาตุมี กี ธาตุเป็นต้น มี นา เป็นวิกรณปัจจัย เมื่อลงแล้วมีอำนาจดังนี้

1. หลักเกณฑ์ทั่วไป

เมื่อลง นา ปัจจัยแล้ว:

  • โดยทั่วไปให้คงไว้:
    ชิ + นา + ติ = ชินาติ
    จิ + นา + ติ = จินาติ
    มุ + นา + ติ = มุนาติ
  • รัสสะสระที่สุดธาตุ (เสียงยาวเป็นสั้น):
    ลุ + นา + ติ = ลุนาติ
    ลี + นา + ติ = ลีนาติ
    ธุ + นา + ติ = ธุนาติ

2. กฎเฉพาะธาตุ

กี ธาตุ (ซื้อ)

  • เมื่อลง นา ปัจจัยแล้ว ให้รัสสะ อี เป็น อิ แล้วแปลง นา เป็น ณา
    กี + นา + ติ → กิ + ณา + ติ = กิณาติ, วิกฺกิณาติ

มา ธาตุ (กะ, ประมาณ)

  • เมื่อลง นา ปัจจัยแล้ว เอาสระที่สุดของ มา ธาตุเป็น อิ
    มา + นา + ติ = มินาติ

ญา ธาตุ (รู้)

ในเพราะ นา ปัจจัย มีการอาเทส ญา ธาตุ ได้หลายรูปแบบ ดังนี้:
  • อาเทส ญา เป็น ชา: (ต้องลง นา ปัจจัยเท่านั้น)
    ญา + นา + ติ = ชานาติ
    ญา + นา + เอยฺย = ชาเนยฺย
  • อาเทส ญา เป็น ชํ: (เฉพาะในที่มี ญา ซึ่งแปลงมาจาก เอยฺย วิภัตติ)
    ให้แปลง ญา ธาตุเป็น ชํ, ลบ นา ปัจจัย
    สำเร็จรูปเป็น ชญฺญา
  • แปลง นา ปัจจัย เป็น ย:
    สำเร็จรูปเป็น ญายติ
  • แปลง ญา ธาตุ เป็น นา:
    สำเร็จรูปเป็น นายติ, นายเร

3. ข้อสังเกตเพิ่มเติม

เฉพาะ กี ธาตุ เป็น "อเนกคณิกธาตุ" สามารถประกอบปัจจัยในหมวด สวาทิ ได้ด้วย
